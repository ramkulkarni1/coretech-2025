## How to Add a New Slide

1. **Create the new slide HTML file** (e.g., `slide8.html`)
   - Copy any existing slide as a template
   - Update the content
   - Make sure it includes:
     ```html
     <link rel="stylesheet" href="common.css">
     <div class="navigation"><!-- Navigation will be generated by navigation.js --></div>
     <script src="navigation.js"></script>
     ```

2. **Add the slide to navigation.js**
   - Open `navigation.js`
   - Add your new slide to the `slides` array:
     ```javascript
     const slides = [
         { file: 'index.html', title: 'Core Tech' },
         { file: 'slide2.html', title: 'What We Offer' },
         // ... existing slides ...
         { file: 'slide8.html', title: 'Your New Slide Title' }  // Add this line
     ];
     ```

That's it! The navigation will automatically update.

## How to Remove a Slide

1. **Delete or rename the HTML file** you want to remove
2. **Remove the entry from navigation.js**
   - Open `navigation.js`
   - Delete the corresponding line from the `slides` array

The navigation will automatically adjust.

## How to Reorder Slides

Simply reorder the entries in the `slides` array in `navigation.js`.

## Available Grid Layouts

The `common.css` file provides several fixed grid layouts:

- `.grid-2x2` - 2 columns × 2 rows (4 items)
- `.grid-3x1` - 3 columns × 1 row (3 items)
- `.grid-3x2` - 3 columns × 2 rows (up to 6 items)
- `.grid-3x3` - 3 columns × 3 rows (up to 9 items)

Use these with `.grid-container`:
```html
<div class="grid-container grid-3x2">
    <div class="card">...</div>
    <div class="card">...</div>
    <!-- ... -->
</div>
```

## Card Types

- `.card` - Standard content card with icon, title, and description
- `.stat-card` - Statistics card with large number
- `.platform-card` - Platform card with icon and count
- `.achievement-card` - Achievement card with colored border and list
- `.theme-card` - Translucent theme card
- `.roadmap-card` - Roadmap card with colored header

## Navigation

- **Keyboard**: Use arrow keys (← →) to navigate between slides, Home key to return to first slide
- **Buttons**: Each slide has Home, Previous, and Next buttons
- **Slide Counter**: Shows current slide number / total slides
  - **Jump to Slide**: Click on the slide number to edit it, enter a slide number (1-12), and press Enter to jump directly to that slide
  - Press Escape to cancel editing

## Using Images and Media

All images and media files should be placed in the `images/` folder. To reference them in your slides:

```html
<!-- For images -->
<img src="images/your-image.png" alt="Description">

<!-- For videos -->
<video src="images/your-video.mp4" controls></video>

<!-- For GIFs -->
<img src="images/your-animation.gif" alt="Description">
```
